# 23' 한국품질재단 빅데이터 분석가 교육과정 3차반 FINAL PROJECT<br> 
## MCI PROGRAM(대상 수상)<br>
### 해당 프로젝트 내 역할
- 팀 로고 디자인, 개발취지 작성
- 주제 선정 및 데이터 확보
- Web 총괄
- 부트스트랩 템플릿을 기초하여 웹 전체적인 로드맵 구상 및 구현
- 플라스크에 부착된 각 테스트 원할하게 연결
- 프론트 오류 수정
- 테스트 결과를 보여주는 대시보드 구상 및 구현
- 챗봇 기능, 음악 삽입, 다운로드 링크 등 사용자 편의를 고려한 다양한 기능 구현
- 컨셉에 맞게 디자인 배치
- Roboflow 를 통한 이미지 라벨링 및 증식
- YOLOv5 모델에 전처리 된 이미지를 학습시켜 정확도 상승
